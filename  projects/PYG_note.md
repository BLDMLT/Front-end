#### 6. logo SEO 优化
1. logo 里面首先放一个h1 标签 -> 提权
2. h1 里面有一个链接，可以返回首页
3. 链接里面放文字 -> 网站名称 - >  不用显示
     + text-indent -> 移动到盒子外
     + 直接 font-size:0 看不见文章 -> 京东做法
4. 最后给链接一个title属性 -> 鼠标放到logo 上就可以看到提示文字
